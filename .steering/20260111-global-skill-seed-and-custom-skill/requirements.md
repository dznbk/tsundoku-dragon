# グローバルスキルの初期データとカスタムスキル登録機能

## 概要

グローバルスキルの初期データ投入と、本登録時の新規スキル自動登録機能を実装する。

## 背景

- 現在、スキルの読み取り機能は実装済みだが、以下が未実装:
  1. グローバルスキルの初期データ投入（シードスクリプト）
  2. 本登録時に新規スキルをカスタムスキルに自動追加
- テーブル作成時にグローバルスキルが空のまま
- 本登録時に新規スキルを入力しても、次回の候補に表示されない
- スキルの重複・表記揺れが発生する可能性がある

## 要件

### 機能要件

- [x] グローバルスキル初期データを投入するシードスクリプトを作成
- [x] 本登録時、入力されたスキルがグローバルスキルに存在しない場合、ユーザーのカスタムスキルに自動登録
- [x] 自動登録されたカスタムスキルが次回の本登録時にオートコンプリート候補として表示される

### 非機能要件

- 既存の `scripts/` パターンに従う
- 既存の Repository/Service パターンに従う

## 受け入れ条件

- [x] `scripts/seed-global-skills.ts` でグローバルスキルが投入できる
- [x] 本登録時に新規スキルがカスタムスキルとして自動登録される
- [x] カスタムスキル登録は重複チェックを行う（既存のグローバル/カスタムスキルは登録しない）
- [x] テストがすべてパスする

## 対象外（スコープ外）

- スキル管理API（管理者によるグローバルスキル追加、カスタムスキル削除、スキルマージ機能）は将来対応
