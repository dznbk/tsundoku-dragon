# 本の登録画面

## 概要

新しい積ん読を「敵（ドラゴン）」として登録し、討伐対象に追加するフォーム画面を実装する。

## 背景

- ユーザーが本を登録することで読書管理を開始できる
- ISBNから書影・ページ数を自動取得し、入力の手間を省く
- スキルタグを紐づけることで、読書による経験値獲得につなげる

## 要件

### 機能要件

- [ ] タイトル（必須）、ISBN（任意）、ページ数（必須）、スキル（任意）を入力できる
- [ ] ISBN入力時にNDL APIから書影URLとページ数を自動取得する
- [ ] 書影をリアルタイムでプレビュー表示する
- [ ] スキルはタグ入力UIで複数選択・新規追加できる
- [ ] スキル入力時にオートコンプリートで候補を表示する
- [ ] 登録成功時に成功メッセージを表示し、フォームをリセットする
- [ ] 続けて別の本を登録できる

### 非機能要件

- レスポンシブ対応（モバイル: 縦積み、タブレット以上: 横並び）
- アクセシビリティ（aria属性、キーボード操作）
- ISBN入力のdebounce（500ms）

## 受け入れ条件

- [ ] 本のタイトルとページ数を入力して登録できる
- [ ] ISBN入力で書影とページ数が自動入力される
- [ ] スキルを選択または新規追加できる
- [ ] 登録後にフォームがリセットされ続けて登録できる
- [ ] モバイル・タブレット・PCで適切に表示される
- [ ] キーボードのみで全操作が可能

## 対象外（スコープ外）

- 本の編集・削除機能（別画面で対応予定）
- 画像アップロード機能
- スキルのカテゴリ表示
