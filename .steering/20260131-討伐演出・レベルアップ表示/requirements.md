# 討伐演出・レベルアップ表示

## 概要

ドラゴン討伐時の演出を実装する。討伐メッセージ、経験値表示、全スキルのレベルアップ表示、ホームへ戻るボタンを順番に表示し、達成感を演出する。

## 背景

- 戦闘システムは実装済みだが、討伐時の演出がない
- 経験値計算・スキル更新ロジックは #56 で実装済み
- BattleMessage コンポーネントは #58 で実装済み（再利用可能）
- 討伐時に獲得経験値やレベルアップを表示することで、ユーザーの達成感を高める

## 要件

### 機能要件

- [x] 「○○ を たおした！」メッセージを表示する
- [x] 「けいけんち ×× ポイント かくとく！」を表示する
- [x] 討伐ボーナスがある場合、「とうばつボーナス ×× ポイント！」を表示する
- [x] レベルアップしたスキルがある場合、順番に表示する
- [x] 全演出完了後に「ホームへもどる」ボタンを表示する
- [x] ボタンクリックでホーム画面に遷移する

### 非機能要件

- DotGothic16（ピクセルフォント）を使用してRPG感を演出
- 既存の BattleMessage コンポーネントを再利用する

## 受け入れ条件

- [x] 討伐メッセージが正しく表示される
- [x] 経験値が正しく表示される
- [x] 討伐ボーナスがある場合のみボーナスメッセージが表示される
- [x] レベルアップしたスキルが全て表示される
- [x] 複数スキルがレベルアップした場合、順番に表示される
- [x] 全演出完了後に「ホームへもどる」ボタンが表示される
- [x] ボタンクリックで onGoHome が呼ばれる
- [x] テストがパスする

## 対象外（スコープ外）

- ドラゴンの討伐エフェクト（アニメーション）
- BGM・効果音
- 戦闘ログAPIへの保存（既に #65 で実装済み）
