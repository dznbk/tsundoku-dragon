# ホーム画面（本一覧）

## 概要

登録した本を一覧表示し、ユーザーの現在の状況を把握できるホーム画面を実装する。「今日はどの本を読もうかな」「どれくらい進んでるかな」を確認する起点となる。

## 背景

- 現在のホーム画面は「ようこそ」メッセージと「本を登録する」ボタンのみ
- 登録した本を確認する方法がない
- ユーザーの読書進捗を一覧で把握できるようにしたい

## 要件

### 機能要件

- [x] 本の一覧をグリッド表示（2列）
- [x] 本カードに書影・タイトル・進捗バー・ステータスバッジを表示
- [x] デフォルトは「戦闘中（reading）」の本のみ表示
- [x] トグルで「討伐済み（completed）」も表示可能
- [x] ステータス表示（ユーザー名、総討伐数、総読了ページ数、上位3スキル）
- [x] 新規登録ボタンから本の登録画面へ遷移

### 非機能要件

- モバイルファースト（2列グリッド）
- SFC時代のRPG風デザイン（ドラクエ風ウィンドウ）
- アクセシビリティ（WCAG 2.1 Level AA）

## 受け入れ条件

- [x] 登録した本が一覧に表示される
- [x] 進捗バーが currentPage / totalPages を反映している
- [x] 討伐済みトグルで completed の本が表示/非表示を切り替えられる
- [x] ステータス表示が正しい値を表示している
- [x] 新規登録ボタンから本の登録画面へ遷移できる

## 対象外（スコープ外）

- 本カードタップ → 詳細画面遷移（詳細画面未実装のため）
- スキル一覧リンク → スキル一覧遷移（スキル一覧未実装のため）
