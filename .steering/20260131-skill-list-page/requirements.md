# スキル一覧画面

## 概要

ユーザーのスキル成長を確認する画面。スキルレベル、経験値バー、ソート・フィルタ機能を実装する。

## 背景

- ユーザーが自分の成長を俯瞰し、達成感を得られる場所が必要
- ホーム画面では上位3スキルのみ表示しており、全スキルを確認できない
- planning/screen-design.md で定義された「スキル一覧画面」の実装

## 要件

### 機能要件

- [ ] スキル一覧の表示（名前、レベル、経験値バー、累計経験値）
- [ ] ソート機能（レベル順/名前順）
- [ ] フィルタ機能（スキル名で検索）
- [ ] ホーム画面からの遷移リンク

### 非機能要件

- 既存の `GET /skills` API を使用（追加API不要）
- 既存のデザインシステム（DQWindow, DQButton等）に準拠
- HpBar と同様のプログレスバーコンポーネントを経験値用に作成

## 受け入れ条件

- [ ] スキル一覧が正しく表示される
- [ ] スキルがない場合、適切なメッセージが表示される
- [ ] レベル順・名前順でソートできる
- [ ] スキル名でフィルタリングできる（大文字小文字区別なし）
- [ ] フィルタ該当なしの場合、適切なメッセージが表示される
- [ ] ホーム画面から遷移できる
- [ ] 戻るボタンでホームに戻れる

## 対象外（スコープ外）

- 最近更新順のソート（Issue仕様では含まれていない）
- スキルの編集・削除機能
- APIの追加実装
