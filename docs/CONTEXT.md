## 決定済み事項サマリ

| 項目 | 決定内容 |
|------|----------|
| ターゲット | 一般書籍含む、一人利用 |
| 認証 | SNS認証（複数デバイス対応のため） |
| スキル構造 | フラット、ユーザー定義可能 |
| スキル選択 | 選択肢から選ぶ or 自由入力で新規追加 |
| レベルアップ | あり |
| 戦闘方式 | 一方的攻撃、ドラクエ風メッセージ |
| 敵画像 | デフォルト1種、複数対応の構造で実装 |
| スキル表示 | MVP時点はリスト表示 |
